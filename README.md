# Growth of a dollar over time

Simple app that calculates growth of a dollar over time.

## Calculate pre tax income

+ Initial amount is `$1`.
+ Period is `5 years`.

`node get-pre-tax`

## Calculate post tax income

+ Initial amount is `$1`.
+ Period is `5 years`.
+ Income tax is `20%`.

`node get-post-tax`
